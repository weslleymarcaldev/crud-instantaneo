# CRUD Instantaneo Free - VS Code Extension (PT-BR)

Repositório público da versão Free da extensão CRUD Instantaneo para VS Code.

## O que a versão Free gera

- Model
- Controller
- Rotas
- SQL básico

Linguagem suportada na Free:

- Node.js

## Estrutura do projeto

- [extensions/crud-instantaneo-free](extensions/crud-instantaneo-free)

## Como usar no VS Code

1. Instale a extensão publicada no Marketplace.
2. Abra um projeto.
3. Execute o comando `CRUD Instantâneo Free: Gerar arquivos`.
4. Informe entidade e campos no formato `nome:string,preco:decimal`.

## Desenvolvimento local

```bash
cd extensions/crud-instantaneo-free
npm install
npm run package
```

## Versão Pro

A versão Pro (Node.js, C# e PHP) é distribuída separadamente em canal privado/comercial.
